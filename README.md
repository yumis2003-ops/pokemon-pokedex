# 初代赤緑風ポケモン図鑑 (Retro Pokémon Pokedex)

PokeAPIを活用した、初代「ポケットモンスター 赤・緑」（ゲームボーイ）のデザインと操作感を徹底的に再現したレトロでプレミアムなWebアプリケーションです。

![画面イメージ](https://raw.githubusercontent.com/yumis2003-ops/pokemon-pokedex/main/images/preview.png) *(※プレビュー画像はイメージです)*

## 🎮 デモの動かし方

1. このリポジトリをクローンします。
   ```bash
   git clone https://github.com/yumis2003-ops/pokemon-pokedex.git
   cd pokemon-pokedex
   ```
2. 静的Webサーバーを起動します（例: Pythonの簡易サーバー）。
   ```bash
   python3 -m http.server 8000
   ```
3. ブラウザで `http://localhost:8000` を開きます。

## ✨ 主なこだわり・プレミアム機能

1. **ゲームボーイ液晶（LCD）の完全再現**:
   - 縦横の極細ピクセルグリッドによる液晶の立体感。
   - スキャンライン（走査線）とガラスベゼルに反射する斜めの光沢エフェクト。
   - `image-rendering: pixelated` による、高精細画像に依存しないドット絵のクッキリとした強調表示。

2. **Web Audio API によるPSG風音源合成**:
   - 外部のmp3等の音声ファイルを使用せず、JavaScriptだけで矩形波や鋸歯状波をリアルタイム合成してPSG音源のようなレトロSEを再生。
   - スクリプト移動音、決定音（ピコーン）、キャンセル音（ピピッ）、エラー音（ブー）、ゲームボーイ起動音を再現。
   - 画面中央の「SOUND」トグルスイッチまたはキーボードの **`S`** / **`START`** で音声を有効化できます。

3. **日本語50音検索 & 爆速一致**:
   - 十字キーとAボタンでカタカタ入力できる50音キーボードパネルを実装。
   - 全1025匹のデータ対照表 (`pokemon_data.js`) を事前にビルドして組み込んでいるため、ひらがな、カタカナ、英語名、図鑑番号での爆速検索が可能。

4. **「たかさ くらべ」機能**:
   - 詳細画面の右側液晶に、主人公（レッド: 1.7m）と選択したポケモンの比較シルエットが表示され、高さに応じてドット絵の表示サイズが動的にスケーリングされます。

## 🎹 操作マッピング

| 動作 / ボタン | キーボード操作 | 画面上の操作 |
| :--- | :--- | :--- |
| **スクロール（リスト）** / **カーソル移動** | `↑` / `↓` / `←` / `→` | 十字キー（D-PAD）のクリック |
| **決定（リスト）** / **文字選択（検索）** | `Enter` または `Z` | **Aボタン** のクリック |
| **戻る（詳細）** / **1文字消去（検索）** | `Backspace` または `X` | **Bボタン** のクリック |
| **検索画面の表示/非表示** | `Space` | **SELECT** のクリック |
| **サウンドのON/OFF** | `S` | **START** または **SOUNDスイッチ** |
| **図鑑番号で直接ジャンプ** | テンキー数字を入力後、`GO` | 右側の物理テンキーをクリック |

## 🛠️ 技術スタック

- **Core**: Vanilla HTML5, Vanilla JavaScript (ES6+)
- **Styling**: Vanilla CSS3 (Custom Variables, Flexbox, CSS Grid)
- **API**: [PokeAPI](https://pokeapi.co/)
- **Audio**: Web Audio API (Realtime Synthesizer)
